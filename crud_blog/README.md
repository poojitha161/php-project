# CRUD Blog Application

## Objective

Develop a PHP and MySQL based CRUD Blog Application.

## Features

* Create Posts
* Read/View Posts
* Update/Edit Posts
* Delete Posts

## Technologies Used

* PHP
* MySQL
* XAMPP
* phpMyAdmin

## Project Structure

* db.php
* create.php
* read.php
* edit.php
* delete.php
* index.php

## Outcome

Successfully implemented CRUD operations using PHP and MySQL.
